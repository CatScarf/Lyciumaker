# Lycium在线三国杀卡牌制作器

请访问 [Lycium三国杀卡牌制作器](https://catscarf.github.io/) 以使用本应用

仅供学习交流。点个小星星支持一下本项目啦~

## 安装方法

```bash
git clone https://github.com/CatScarf/Lyciumaker.git
cd Lyciumaker
npm install
npm run dev
```
